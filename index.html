How to Create a New Webpage with Github

Step One- Create a Repository

Click on the + symbol in the upper right-hand corner of the screen. Then select "New repository".

Step Two- Name Repository

Name the repository with (name).github.io, and check the box that says "Initialize this Repository with README."

Step Three- Upload Files

Click on "Upload Files", then drag and drop files. After the files have been uploaded, click "Commit Changes".

Step Four- Change Source to Master Branch

Go to settings, then scroll down to the GitHub Pages box. Click on the box that says "None" and select "master branch".

Step Five- Go to Your New Website

Click on you new link in the green box to view your new website.
